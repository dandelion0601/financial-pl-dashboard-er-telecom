# financial-pl-dashboard-er-telecom
Управленческий отчёт P&amp;L с KPI и дашбордами на основе данных Еr-Теleсoм (анонимизировано)

Автоматизированный управленческий отчёт на основе данных из SAP, построенный в Excel с использованием Power Query и Power Pivot. Проект направлен на объединение разрозненных источников данных и предоставление единой картины финансовой эффективности по сегментам, продуктам и проектам.
>  Все данные обезличены. Реальные суммы изменены для сохранения конфиденциальности.

#Цель проекта:
Создать единый источник для управленческой отчётности, позволяющий:
- Сравнивать фактические показатели с бюджетом и планом
- Оценивать результаты KPI и управлять его составляющими
- Анализировать рентабельность по сегментам (B2B, B2G, B2R, B2O)
- Оценивать эффективность проектов и видов деятельности
- Сократить время подготовки отчёта с 3 дней до <1 часа

## Подход и технологии
### 1. **Источники данных**
Объединено 10+ выгрузок из SAP (через надстройку Analysis) в единую модель:
- Выручка (Финальные источники: `Выручка_продажи`, `Источник_Выручка`, Исходники: 10+ файлов )
- Затраты (Финальные источники: ‘Источник’ , Исходники : 10+ файлов с `ОPEX/CAPEX`)
- Среднесписочная численность (ССЧ)
- Справочники (`Календарь`, `Сегмент Меппинг`, `Вид статьи Меппинг`, `Проект имя`, `Показатель Меппинг`, `БЕ меппинг`, `МРФ Меппинг` и др.)

### 2. **Data Modeling (Power Pivot)**
Построена реляционная модель данных с 16 таблицами и связями «один ко многим»:

![Схема модели данных](model_schema.png)  

- **Главная таблица**: `Источник` — содержит все факты и планы по затратам и выручке.
- **Справочники**: обеспечивают фильтрацию по сегментам, проектам, видам статей, БЕ, МРФ и др.

### 3. **Расчёт метрик (DAX)**
На основе модели рассчитаны ключевые KPI:

| Метрика | Формула |
|---------|---------|
| `% Выручка 25Ф-24Ф` | `DIVIDE([Sum_выручка 25 ф];[Sum_выручка 24ф])-1` |
| `% доли затрат в выручке` | `DIVIDE([Доля затрат в выручке 25f];[Доля затрат в выручке план 25]) - 1` |
| `НФЗП факт 25` | `DIVIDE([НФЗП факт 25];[Sum_выручка 25n])` |
| `ПТ план 25` | `DIVIDE([ПТ Факт 25];[ПТ план 25])` |
| `Откл ф-п ФОТ 25` | `[Sum_ФОТ факт 25] - [Sum_ФОТ план 25]` |

>  Полный список мер и их логика доступны в файле `measures.xlsx` или по кнопке "Управление мерами" в Excel.

### 4. **Визуализация (Excel Dashboards)**
Созданы интерактивные дашборды с возможностью фильтрации по:
- Сегментам (B2F, B2G, B2O, B2R)
- Проектам и группам проектов
- Видам статей (ОПЕХ, CAPEX, ФОТ, прочие)
- Периодам (месяц, квартал, год)

## Ключевые выводы и бизнес-эффект

### Что удалось достичь:
- **Автоматизация**: Ручной сбор и сводка данных заменены на автоматическую модель → **сокращение времени подготовки на 80%**.
- **Прозрачность**: Отделы производства получили возможность самостоятельно анализировать свою P&L и KPI по сегментам и проектам.
- **Глубина анализа**: Возможность детализировать показатели до уровня отдельных проектов и видов статей.
